# 🎙️ Fono IA

> **Análise de voz em tempo real e feedback inteligente com IA.**

O **Fono IA** é uma plataforma web *mobile-first* projetada para transformar a maneira como a fala é analisada. Utilizando modelos avançados de inteligência artificial, o sistema captura áudio, processa dados fonéticos e entrega feedbacks instantâneos, unindo tecnologia de ponta a uma interface minimalista e intuitiva.

---

## ✨ Funcionalidades Principais

* **🎙️ Reconhecimento de Voz High-End:** Captura de áudio de alta precisão otimizada para dispositivos móveis.
* **🤖 IA de Feedback em Tempo Real:** Processamento imediato para orientações fonéticas ou comportamentais.
* **📱 Design Ultra-Minimalista:** Interface limpa em tons de preto e branco, utilizando a tipografia **Poppins**.
* **📊 Dashboard de Evolução:** Acompanhamento métrico do progresso do usuário.
* **⚡ Performance de Ponta:** Build ultra-rápido utilizando Vite.

## 🛠️ Stack Técnica

* **Core:** [React.js](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Estilização:** Tailwind CSS (Foco em minimalismo e alto contraste)
* **Tipografia:** Poppins (Google Fonts)
* **IA/Voz:** Integração com APIs de reconhecimento de fala e processamento de linguagem natural.
* **Design:** Prototipagem via Figma.

## 🚀 Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/fono-ia.git](https://github.com/seu-usuario/fono-ia.git)
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
4.  **Acesse:** `http://localhost:5173` (porta padrão do Vite)
